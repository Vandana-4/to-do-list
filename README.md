# 📝 To-Do List Web App

A simple and interactive To-Do List application built using HTML, CSS, and JavaScript. This project helps users manage their daily tasks in an easy and efficient way.

## ✨ Features
- ➕ Add new tasks  
- ✔️ Mark tasks as completed  
- 🗑️ Delete tasks  
- 🎯 Simple and user-friendly interface  
- 📱 Responsive design for better experience on all devices  

## 🛠️ Tech Stack
HTML, CSS, JavaScript
